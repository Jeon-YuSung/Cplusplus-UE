## 5장 반복문

반복문에 대해 이해한다 <br>
while, for, do-while, 중첩 반복문에 대해 배운다 

-------------
### 4.1 반복문의 개요 

반복문의 필요한 이유 : 
+ 조건 분기를 사용하여 프로그램을 100번 실행할 수 있으나, 많은 시간이 소요됨<br>
+ 하나의 프로그램 내부에 조건 분기를 100번 사용하고 프로그램을 1회만 실행할 수 있음<br>
하지만 코드가 굉장히 길어지고, 조건이나 수가 바뀌면 프로그램을 다시 변경해야함<br>
+ 코드를 정해진 만큼 반복할수 있는 반복문(Repetition Statement)을 사용해서 프로그램을 작성할 수 있음

즉, 대부분의 프로그램에서 코드 문장을 반복해야하는 경우가 많음<br>
C++은 코드를 반복할 때 사용하는 구문으로 while문, for문, do-while문을 제공함

### 후위 표현식과 전위 표현식 

반복문을 구성할 때는 반복 횟수를 확인하고 적절한 시점에 종료할 수 있게, Counter를 만들어야함 <br> 
후위증가, 전위증가식을 제공<br>
lvalue++ <br>
lvalue-- <br>
++lvalue <br>
--lvalue <br>

**차이**

부가 작용 (변수의 값)에만 주목하면 둘 다 완전히 같은 결과가 나옴 <br>
차이점은 리턴값인데, 후위 표현식은 원래 값을 리턴하며, 전위 표현식은 부가 작용이 일어난 뒤에 값을 리턴함 <br>
그렇기 때문에, counter로 사용할 때는 두 표현식에 차이가 없다고 생각하고 써도 됨.<br>
일반저그올 대부분은 후위표현식을 사용함 

#### 반복문의 종류 
C++에서의 반복문은  while문, for문, do-while문로 3가지가 있음<br>
while, for 반복문 : 조건을 먼저 확인<br>
do-while 반복문 : 반복문 조건을 나중에 확인. 

---------------------
### 4.2 while 반복문 
while 반복문은 조건을 반복하여 평가하고, 참이면 문장 실행, 거짓이면 반복을 종료후 반복문 이후의 문장들을 실행. <br>
while(조건식){

문장;

}

while문의 조건이 항상 참이면 무한 반복함으로, 조건을 거짓으로 만들어서 반복문을 벗어날 수 있도록 해야함<br>
이를위해 카운터 제어와 이벤트 제어를 활용 

+ **카운터 제어**

반복을 몇 번 해야 하는지 알고 있을 때 사용 <br>
반복문에 진입하기 전에 카운터를 초기화함. <br>
그 다음 반복문 내에서 이 값을 증가 또는 감소를 시킨 뒤, 조건을 활용해서 반복을 종료하게 만드는 방법  <br>

+ **이벤트 제어**

  여러 번 반복하는데, 어떤 상황에서 멈춰야 하는지만, 알고 몇 번 반복해야 할지 모르는 경우 <br>

+ Sentinel 제어 while 반복문  <br>
권한이 없는 사람이 특정한 곳에 들어가지 못하게 함  <br>
데이터처리에서 센티넬은 처리 중지를 나타내기 위해 데이터 리스트에 존재하는 특별한 항목임.  <br>
일반적으로 센티넬은 리스트의 다른 항목과 같은 자료형을 갖지만, 값은 특별한 의미를 갖고 있어야 함.  <br>
즉, while 반복문의 조건에 센티넬을 활용하면, 센티넬을 만나는 순간 반복문이 종료 됨  <br>

+ EOF 제어 while 반복문  <br>
키보드와 파일로부터 데이터를 읽어 들일 때, "더 이상 키보드로부터 입력된 데이타가 없다"와 "파일의 끝부분에 도달했습니다" ㄹ하는것을 알리기 위한 센티넬,  End of File라고 함  <br>
예를 들어 유닉스에서는 ctrl+d, 윈도우 에서는 ctrl + z임  <br>
파일을 데이터 소스로 활용할 때는 별도의  처리가 필요 없지만, 파일을 생성할 때는 자동으로 EOF 맠가 추가 됨. <br>
EOF를 사용하려면 ">>" 연산자를 입력. <br>
">>" 연산자는 스트림에서 항목을 읽다가 eof를 만나면 EOF를 읽어 들이지 않고 단순히 FALSE를 리턴함 (반복문 종료) <br>
EOF가 아닌 항목을 만나면 부가 작용을 일으키고 TRUE를 리턴함 (반복문 실행)  <br>

+ 플래그 제어 while 반복문 <br>
센티넬은 리스트의 마지막 항복에 배치 됨. 하지만 중간에 반복문을 종료하고 싶을 수 도 있음  <br>
플래그는 불 변수의 다른 표현 <br>
반복문에 들어가기 전에 플래그를 false로 설정하고, 반복문 내부에서 어떤 조건이 발생할 때, 이를 true로 변경하면서 조건을 활용 <br>
하지만 조건이 발생하지 않을 경우를 대비해서 다른 제어 매커니즘과 함께 사용해야함.  <br>
예를 들어 플래그 제어 + 센티널 조건을 결합하는 상태로 사용  <br>

#### while 반복문 사용시 주의 사항

1. 반복하지 않은 반복문 <br>
2. 무한하게 반복하는 반복문 <br>
3. 본문이 없는 반복문 <br>
4. 조건으로 부동 소수점을 사용하는 반복문
